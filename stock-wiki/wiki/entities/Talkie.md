# Talkie / 星野

## 概述

Talkie（国际市场）/ 星野（中国国内市场）是[[MiniMax]]推出的全模态AI社交互动平台。用户可以与AI角色进行文本、语音、图片等多模态交互。Talkie是MiniMax收入贡献最大的产品，也是其用户增长的主要驱动力。

## 核心功能

- **AI角色互动**: 与自定义AI角色进行深度对话
- **全模态交互**: 文本 + 语音 + 图片多模态
- **角色创建**: 用户可创建和分享自定义AI角色
- **社交属性**: 类似社交App的体验，具有社区和分享功能
- **订阅增值**: 付费解锁高级功能（更多互动次数、优质模型体验等）

## 商业表现

| 指标 | 数据 |
|------|------|
| 9M 2025收入 | USD 1,875万 |
| 占MiniMax总收入 | 35.1%（第一大收入来源） |
| 付费用户 | 在MiniMax各产品中占主要份额 |

## 用户增长

- Talkie/星野是MiniMax累计2.12亿用户的核心来源
- 在全球200+国家/地区有用户
- 月活用户约2,764万（9M 2025）
- 用户粘性强，互动频次高

## 重要事件

### Apple App Store下架事件
- **时间**: 2024年12月至2025年2月（约2个月）
- **原因**: 在部分司法管辖区的App Store被暂时移除
- **影响**: 管理层认为系内容审核问题，已重新上架
- **风险提示**: AI内容生成产品的分发渠道依赖风险

## 竞争格局

AI社交/角色扮演是一个快速增长的细分赛道：
- **Character.AI**: 美国市场领军者，2024年被Google收购（授权协议）
- **Talkie**: 全球化布局，在中国和新兴市场有优势
- **星野**: 中国市场版本，本土化运营

## 商业模式

1. **订阅服务**: 付费会员享受更多互动次数、优质模型、高级功能
2. **应用内购买**: 虚拟积分，用于消费性/非消费性物品
3. **在线广告**: 在特定AI应用上的广告中介（少量）
4. **数据飞轮**: 用户交互数据反哺模型训练，提升产品体验
