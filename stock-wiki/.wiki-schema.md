## Directory Structure
```
stock-wiki/
├── raw/              # 原始素材（招股书PDF转Markdown）
├── wiki/
│   ├── entities/     # 实体页面（公司、产品、人物、模型）
│   ├── topics/       # 主题页面（行业分析、监管框架）
│   ├── comparisons/  # 对比分析页面
│   ├── synthesis/    # 综合分析页面
│   └── sources/      # 素材来源摘要
├── index.md          # 内容索引
└── .wiki-schema.md   # 本文件
```

## Page Types
- **entities**: 公司、产品、模型、人物的详细实体页面（最低1500字）
- **topics**: 行业主题知识体系（最低5个核心观点）
- **comparisons**: 多维度对比分析
- **synthesis**: 综合研判与投资分析
- **sources**: 原始素材的结构化摘要

## Conventions
- 所有页面使用 Markdown + YAML frontmatter
- 页面间使用 Obsidian 双向链接 `[[Page Name]]`
- 链接名称必须与文件名完全匹配（不含 .md）
- 金额默认以人民币计价，美元金额标注 USD
- 数据截止日期：2025年9月30日（MiniMax）/ 2025年6月30日（智谱）
