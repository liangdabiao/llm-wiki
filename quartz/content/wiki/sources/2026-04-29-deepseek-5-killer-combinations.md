# DeepSeek 5 大王炸工具组合：PPT、视频、图片、会议、3D 全流程

> 深度求索与 5 款主流工具的组合应用，覆盖日常办公高频场景，附详细实操方法

## 核心观点

1. **组合式 AI 工作流效率倍增**：DeepSeek 不是单独使用，而是作为核心大脑与专业工具组合，效果远超单工具使用

2. **五大黄金组合覆盖全场景**：
   - PPT 制作：DeepSeek + Gamma
   - 视频批量生产：DeepSeek + 剪映
   - AI 绘画提示词生成：DeepSeek + Midjourney/即梦
   - 会议纪要整理：DeepSeek + 飞书妙记
   - IP 角色 2D→3D 全流程：DeepSeek + Midjourney + 腾讯混元 3D

3. **行业重塑判断**："几乎所有传统的行业，都可以用 deepseek 重做一遍。或者是优化某个环节，或者是取代某个程序。"

## 实操内容保留

### 组合一：DeepSeek + Gamma（PPT 制作）

**Prompt 模板**：
```
我是一个[身份]，现在需要给[受众]讲解[主题]，帮我设计一个 PPT 大纲，以 markdown 格式输出，要求通俗易懂
```

**操作流程**：
1. DeepSeek 生成 Markdown 格式大纲
2. 打开 Gamma（gamma.app）
3. 选择"新建"→"粘贴文本"
4. 选择主题一键生成炫酷 PPT

### 组合二：DeepSeek + 剪映（图文视频）

**Prompt 模板**：
```
你现在是一个爆款文案专家，现在帮我生成一个文案，主题是[视频主题]，要求字数 500 字左右，[风格要求]
```

**操作流程**：
1. DeepSeek 生成 500 字左右文案
2. 打开剪映→选择"AI 图文成片"
3. 粘贴文案自动生成视频
4. 产能：按此工作流"一天做 100 条视频也不是不可能"

### 组合三：DeepSeek + Midjourney/即梦（AI 绘画）

**Prompt 模板**：
```
我想用 MJ 画一个[画面描述]，帮我生成一下提示词，描述要尽可能详细
```

**核心价值**：
- DeepSeek 自动生成长且专业的英文提示词
- 解决"提示词太长写不出来"的痛点
- 国内用户可转用即梦生成水墨风等中国特色效果

### 组合四：DeepSeek + 飞书妙记（会议纪要）

**Prompt 模板**：
```
这是我的一份会议纪要，帮我总结一下内容要点，提炼核心结论。可以加一些表情和符号，让内容看起来更美观
```

**操作流程**：
1. 飞书妙记录制会议 → 导出文字记录
2. DeepSeek 上传文件 → 生成结构化总结
3. 直接发群交付

### 组合五：DeepSeek + Midjourney + 腾讯混元 3D（IP 设计）

**Prompt 模板**：
```
我想画一个[IP形象描述]，要求[风格要求]。因为我后期想把图片转为 3D 形象，为了避免影响，背景需要是纯白色的。根据以上要求，帮我设计一个 midjourney 的提示词
```

**操作流程**：
1. DeepSeek 生成专业 Midjourney 提示词
2. Midjourney 生成 2D 形象
3. 腾讯混元 3D（3d.hunyuan.tencent.com）图生 3D → 一键生成模型
