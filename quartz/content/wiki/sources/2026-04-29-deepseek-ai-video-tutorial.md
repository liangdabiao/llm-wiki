## 实操内容保留

### 分镜脚本生成 Prompt

**第一轮（基础框架）**：
```
我想做一个关于[主题]的AI视频，帮我设计一下脚本、文案、画面，包含时间轴，以表格的形式输出，视频大约[时长]秒
```

**第二轮（细节强化）**：
```
每一个画面描述写具体一些，主体、细节等都要包含，符合[生图工具名称]生图的语法
```

**示例（普洱茶 30 秒视频）**：
```
我想做一个关于普洱茶叶的AI视频，帮我设计一下脚本、文案、画面，包含时间轴，以表格的形式输出，视频大约30秒
```

### 完整工作流三步法

**Step 1 脚本生成**：
- 打开 chat.deepseek.com
- 开启深度思考模式
- 两轮 Prompt 生成专业分镜表（先出框架，再加细节）

**Step 2 图片 → 视频生成**：
- 图片生成：即梦
- 视频生成工具三选一：
  - 可灵（快手）：https://klingai.kuaishou.com/
  - VIDU
  - 海螺
- 逐张导入图片生成动态视频

**Step 3 后期剪辑**：
- 导入剪映视频轨道
- 添加字幕、音乐、转场
- 导出成品
