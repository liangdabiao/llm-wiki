# DeepSeek装进VSCode，编程非常丝滑！

## 核心信息

使用 Roo Code 插件接入 DeepSeek API，支持 Ollama 本地模型。

### 配置步骤：

1. VSCode 扩展商店搜索 Roo Code 安装
2. 左侧活动栏小火箭图标，选择 DeepSeek，填写 API Key
3. API Key 申请：https://www.deepseek.com

### 问题与替代方案

问题：DeepSeek API 太火爆，使用不稳定
