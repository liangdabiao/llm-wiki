# 鱼皮的 AI 指南 - 2、AI 实用工具

AI 相关工具百花齐放，几十个主流工具覆盖 AI 对话、写作、图像、视频、音频、办公、编程等各个应用场景。

不必都去学习。下面着重分享程序员日常工作中非常实用的一整套工具链。

更多内容：免费开源 AI 知识库 https://github.com/liyupi/ai-guide

---

## 一、开发类工具

### 1. 日常问题解决与知识搜索 - AI 对话产品

**适用场景**：简单问题、内容搜索、知识查询。

**推荐产品**：
- 国产：DeepSeek、字节跳动 元宝、豆包、通义千问、文心一言、Kimi
- 海外：GPT、Claude

**使用技巧**：
- 同时打开好几个网站一起用，挑选最合适的回答
- 嫌麻烦可以用聚合 AI 应用：https://botgroup.chat/ 一个聊天框和所有主流大模型对话

### 2. 小项目与新项目开发 - Cursor

**适用场景**：小项目、新项目快速从零开发。

**特点**：
- 专为编程打造的 AI 工具
- Agent 智能体模式
- 新建文件夹 → 打开 AI 对话框 → 输入提示词 → 等待 → AI 自主生成完整网站
- 真的一行代码都不用写！

**影响**：为什么现在突然有了那么多奇奇怪怪大开脑洞的应用？因为人人都能搞网站了。

### 3. 老项目与大项目维护 - AI IDE 插件

**适用场景**：老项目维护、大型项目开发，AI 作为辅助。

**为什么不用 Cursor**：大型项目上下文太多，AI 难以全部理解，插件更专注辅助。

**推荐插件**：
- 智谱 CodeGeex
- JetBrains Assistant
- 通义灵码
- GitHub Copilot

**核心功能**：
- 以项目代码为上下文信息
- 生成补全代码
- 分析错误信息，自动改 Bug
- 解释代码
- 生成单元测试
- 生成 Git 提交信息
- 全部在 IDE 内完成，不用来回跳转

### 4. 其他开发场景特殊工具

#### 数据库管理 - Chat2DB

网址：https://chat2db-ai.com/

功能：一句话完成数据管理、SQL 生成、数据分析和可视化。

#### API 接口测试 - PostMan

网址：https://www.postman.com/

功能：借助 AI 快速生成不同场景下的测试用例和接口文档。

#### 终端操作 - Warp AI

网址：https://www.warp.dev/warp-ai

功能：自动把自然语言转换为正确的命令并执行，不用记忆复杂命令。

## 二、写作类工具

程序员工作除了开发之外，也离不开写文档和作图。

### 1. 写作与文档编写 - 语雀 AI

**特点**：
- 团队协作文档首选
- 功能丰富：Markdown 语法写文章、嵌入画板、一键进入演讲模式
- 内嵌 DeepSeek 大模型
- AI 助手功能：完善丰富内容、改写口吻、总结转写内容

### 2. 作图 - 豆包 + Mermaid

程序员写技术文档必备：流程图、架构图让文档更好理解。

**流程**：
1. 把作图信息提供给豆包或其他 AI
2. 让 AI 生成 Mermaid 文本作图语法
3. 下载图片或放到 Mermaid 渲染工具中

**未来趋势**：Gemini 已原生支持多模态能力，可直接生成图文混排文章。未来可能根据代码直接生成有图有文的技术方案。
