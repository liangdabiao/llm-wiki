# DeepSeek 三大文体写作技巧：公文、新闻、广告文案

> 深度解析 DeepSeek 在公文写作、新闻报道、广告文案三大领域的应用技巧与策略

## 核心观点

1. **公文写作三要素**：指定公文类型 + 明确写作目标 + 提供详尽背景信息 + 设定格式要求，四者结合才能产出高质量公文

2. **新闻报道三原则**：主题明确 + 客观中立 + 引用权威数据，根据受众调整语言风格（普通读者活泼 vs 行业专家正式）

3. **广告文案三策略**：明确文案目标 + 输入产品关键词 + 数据驱动市场调研，根据宣传目的灵活调整语言风格

## 实操内容保留

### Prompt 模板

**公文写作**：
```
请撰写一份[公文类型：通知/请示/报告/函]
主题：[具体主题]
背景信息：[详细背景资料]
格式要求：包含引言、主体、数据分析
结构要求：[具体结构要求]
```

**新闻报道**：
```
请撰写一篇关于[新闻主题]的[新闻类型]报道
背景信息：[相关趋势和背景]
要求：保持客观、引用权威数据、避免主观情绪
受众：[普通读者/行业专家]，语言风格对应调整
```

**广告文案**：
```
目标：吸引[目标群体]购买[产品名称]
产品关键词：[核心卖点列表]
目标渠道：[社交媒体/正式发布会等]
要求：结合市场调研数据，语言风格匹配渠道
```

### 操作步骤

1. **公文写作四步法**：指定类型 → 明确目标 → 提供背景 → 设定格式
2. **新闻创作三原则**：主题先行 → 客观表述 → 数据支撑
3. **广告文案三策略**：目标明确 → 关键词精准 → 数据驱动

## 原文精彩摘录

> "在公文写作中，DeepSeek的潜力尤为显著。用户可以通过清晰地指定公文类型（如通知、请示、报告、函等）来激活AI的能力，从而提高写作效率。"

> "在言辞方面，用户应推动DeepSeek保持客观，避免主观情绪的渗透。同时，准确性和可靠性也至关重要。通过要求AI引用权威数据或专家观点，可以增强新闻报道的说服力和权威性。"

> "在广告文案的撰写上，DeepSeek展现了更加灵活和富有创意的一面。首先，明确文案目标是关键。例如，'吸引年轻消费者购买新款智能手表'这样的核心目标，能够引导AI生成更具针对性的内容。"
