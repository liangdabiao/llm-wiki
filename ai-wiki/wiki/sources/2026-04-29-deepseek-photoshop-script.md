# DeepSeek 一句话生成 Photoshop 脚本：三秒日式小清新修图

> DeepSeek 自动生成 JSX 脚本实现一键修图，零代码基础也能快速上手，思路打开 AI 打工新姿势

## 核心观点

1. **代码生成解决自动化痛点**：不需要懂代码，用自然语言描述想要的修图风格，DeepSeek 直接生成可执行的 Photoshop 脚本

2. **三秒出效果效率惊人**：脚本导入 PS 后一键执行，"原片秒变日杂封面既视感"，手动调图需要几十分钟的工作三秒完成

3. **思路价值大于案例本身**：日式小清新只是一个例子，这个方法可以扩展到任何修图风格、任何可脚本化的重复性工作

## 实操内容保留

### Prompt 模板

```
请帮我写 Photoshop 脚本，模仿[风格名称]风格，[具体调参描述]，我的版本是[PS版本号]版
```

**示例**：
```
请帮我写Photoshop脚本，模仿日式小清新风格，室内暖调，我的版本是2025版
```

### 操作流程四步走

❶ **复制代码**：DeepSeek 生成的脚本代码复制进记事本 📝

❷ **格式转换**：记事本 → 格式 → 转纯文本 ‼️

❸ **编码保存**：保存时选择 utf-8 编码

❹ **改后缀名**：把 `.txt` 改成 `.jsx` 后缀（弹窗警告点确定即可）

**PS 执行**：
```
导入照片 → 文件 → 脚本 → 选中制作好的 .jsx 文件 → 执行
```

### 避坑指南（必看）

✔️ 记事本一定要转纯文本格式（否则可能出现编码问题）
✔️ 编码选错会变乱码，必须用 UTF-8
✔️ 脚本后缀必须是 `.jsx`，不是 `.js` 或其他
