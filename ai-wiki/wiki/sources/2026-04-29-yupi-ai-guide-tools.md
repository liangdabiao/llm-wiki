## 一、开发类工具详解

### 1. 日常问题：AI 对话产品矩阵

**适用场景**：简单问题解决、知识搜索、快速咨询。

**推荐产品**：
- 国产优先：**DeepSeek**、字节跳动 元宝、豆包、通义千问、文心一言、Kimi
- 海外：GPT、Claude

**高级技巧**：
- 多开对比：同时打开多个网站，挑选最合适的回答
- 聚合工具：https://botgroup.chat/ 一个聊天框对话所有主流大模型

### 2. 从零开发：Cursor

**适用场景**：小项目、新项目从零快速开发。

**核心能力**：
- 专为编程打造的 AI IDE
- Agent 智能体模式
- 一句话指令生成完整网站
- 真的一行代码都不用写！

**行业影响**：为什么现在突然有了那么多奇奇怪怪大开脑洞的应用？因为人人都能搞网站了。

### 3. 大项目维护：AI IDE 插件

**适用场景**：老项目维护、大型项目开发，AI 作为辅助而非主导。

**为什么不用 Cursor**：大型项目上下文太多，AI 难以全部理解，插件辅助模式更合适。

**推荐插件**：
- 智谱 CodeGeex
- JetBrains Assistant
- **通义灵码**（国产推荐）
- GitHub Copilot

**通用功能**：
- 以项目代码为上下文
- 智能代码补全
- 自动分析错误信息并改 Bug
- 代码解释
- 单元测试自动生成
- Git 提交信息自动生成
- IDE 内完成一切，不用来回跳转

### 4. 专业工具补充

| 工具 | 用途 | 核心价值 |
|------|------|---------|
| **Chat2DB** | 数据库管理 | 一句话完成数据管理、SQL 生成、数据分析、可视化 |
| **PostMan AI** | API 接口测试 | AI 快速生成不同场景测试用例和接口文档 |
| **Warp AI** | 终端操作 | 自然语言自动转换为正确命令执行，不用记复杂命令 |

## 二、写作类工具

### 1. 团队协作文档：语雀 AI

**核心能力**：
- Markdown 语法支持
- 嵌入画板
- 一键演讲模式
- **内嵌 DeepSeek 大模型**
- 内容丰富完善、改写口吻、总结转写

### 2. 技术文档作图：豆包 + Mermaid

**流程图、架构图自动化生成流程**：
1. 把作图信息提供给豆包或其他 AI
2. AI 生成 Mermaid 文本语法
3. 下载图片或放入渲染工具

**未来趋势**：Gemini 已原生支持多模态图文混排，未来可能根据代码直接生成完整有图有文的技术方案。

## 实操内容

### 开发类工具选择指南

| 场景 | 推荐工具 | 核心优势 |
|------|---------|---------|
| 日常问题解决 | DeepSeek / 豆包 / Claude | 多开对比，挑选最合适回答 |
| 新项目从零开发 | **Cursor** | Agent 模式，一句话生成完整网站 |
| 大型项目维护 | 通义灵码 / GitHub Copilot | 以项目代码为上下文，辅助补全 |
| 数据库管理 | Chat2DB | 一句话完成数据管理、SQL 生成 |
| 终端操作 | Warp AI | 自然语言自动转换为正确命令 |

### Mermaid 作图工作流

1. 把作图信息提供给豆包或其他 AI
2. AI 生成 Mermaid 文本语法
3. 下载图片或放入渲染工具（支持：Typora、VS Code、Obsidian、GitHub）
