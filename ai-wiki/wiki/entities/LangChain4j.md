# LangChain4j

> Java 版本的 LangChain，专为智能体开发设计的完整工具链

## 简介

LangChain4j 是 LangChain 生态的 Java 实现版本，提供了大量现成的方法来提高 AI 应用开发效率。相比 Spring AI，它更加灵活，特别适合开发复杂的智能体系统。

## 核心特性

### 1. 完整的 Agent 工具链
- 任务规划模块
- 工具调用框架
- 多轮对话管理
- 记忆系统实现

### 2. 丰富的组件生态
- 多种文档加载器
- 向量数据库连接器
- 链式调用 DSL
- 输出解析器

### 3. 灵活的架构设计
- 模块化设计，按需组合
- 可扩展的插件机制
- 与 Spring AI 可混合使用
- 支持自定义中间件
